# Лифт со слогами

**[Играть онлайн](https://ivanalekseevichpopov.github.io/elevatorGame/)**

Обучающая игра для детей 5 лет. Лифт с согласной буквой ездит по этажам с гласными, формируя слоги.

## Структура проекта

```
elevatorGame/
├── index.html           - HTML структура (экраны выбора, игры, победы)
├── styles.css           - Стили (здание, лифт, анимации, эффекты)
├── game.js              - Логика игры (управление, состояние, победа)
├── syllables-words.json - Слова и иконки для каждого слога
└── icons/               - SVG иконки из Lucide (65 файлов)
```

## Механика игры

1. Игрок выбирает согласную букву (М, Н, П, Б, К, Т, Д, С, Л, Р)
2. Лифт с выбранной буквой стартует на 1 этаже (А)
3. Управление: кнопки ▲/▼ или стрелки клавиатуры
4. 10 этажей с гласными: А, О, У, И, Ы, Э, Я, Ё, Ю, Е
5. 11-й этаж - финиш (крыша с флагом)
6. При каждой остановке показывается слог (согласная + гласная)
7. Достижение крыши = победа (звук + конфетти)

## Визуальные элементы

- Треугольная крыша с трубой
- Окна на этажах
- Балконы на этажах 2, 5, 8
- Входная дверь на 1 этаже
- **Индивидуальные цветовые схемы** для каждой согласной буквы
- **Font Awesome**: иконки для интерфейса (подключены через CDN)

## Цветовые схемы

Каждая согласная имеет уникальную цветовую палитру:

| Буква | Цвет |
|-------|------|
| М | Изумрудный |
| Н | Синий |
| П | Фиолетовый |
| Б | Индиго |
| К | Коралловый |
| Т | Оранжевый |
| Д | Золотистый |
| С | Бирюзовый |
| Л | Лавандовый |
| Р | Мятный |

Цвета применяются через CSS-переменные (`--color-dark`, `--color-primary`, `--color-light`, `--color-lighter`) и влияют на: кнопки выбора, лифт, шахту, панель слога, кнопки управления, экран победы и конфетти.

## Ключевые переменные (game.js)

- `vowels` - массив гласных букв
- `consonants` - массив согласных для выбора
- `colorSchemes` - объект с цветовыми схемами для каждой буквы
- `selectedConsonant` - выбранная согласная
- `currentFloor` - текущий этаж (0-10)
- `collectedSyllables` - собранные слоги для экрана победы

## Ключевые функции (game.js)

- `initSelectionScreen()` - создание кнопок выбора согласных
- `selectConsonant(letter)` - выбор буквы и старт игры
- `applyColorScheme(letter)` - применение цветовой схемы через CSS-переменные
- `initBuilding()` - построение здания с этажами
- `moveUp() / moveDown()` - движение лифта
- `updateSyllableDisplay()` - обновление отображаемого слога
- `showVictory()` - экран победы
- `playVictorySound()` - звук победы (Web Audio API)
- `createConfetti()` - анимация конфетти
